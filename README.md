Título do projeto: Viva Bem 
Status do projeto (em desenvolvimento): Completo
Tecnologias que serão aplicadas (considere apenas as tecnologias abordadas no curso): Linguagem Java e Banco de dados MYSQL
Time de desenvolvedores: Camily Ramos
Objetivo do software (a que deve servir): O objetivo do software é levar praticidade para a vida das pessoas que buscam um amparo psicologico ou melhorar com sigo mesmas.
Funcionalidades do sistema (requisitos): O sistema possui um diario de emoções que fica salvo em um arquivo de texto, para que possa ser apresentado para o terapeuta para ajudar no controle de 
diagnosticos e juntamente com uma planilha que mostre os profissionais disponiveis e seus contatos.
